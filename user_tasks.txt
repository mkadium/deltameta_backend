# DELTAMETA - User Tasks

Use this plain text file to give the assistant tasks in normal English.

Guidelines:
- Write one task per line or paragraph.
- Be specific about desired outcomes, files to change, or constraints.
- If the task involves code, include filenames, function names, or code snippets where helpful.

Example tasks:
- "Create a React app scaffold in this repo with Vite and TypeScript."
- "Update `project_overview.md` to add a new 'Roadmap' section summarizing Q1 goals."
- "Refactor `src/utils/api.ts` to use retry logic and add tests."

When you save or update this file and tell the assistant to "perform tasks" or "start tasks", the assistant will read the entries and ask clarifying questions if needed.
