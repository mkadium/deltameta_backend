# Telemetry Logs Directory

This directory contains daily telemetry log files generated by OpenTelemetry instrumentation.

## File Format

Files are named: `telemetry_YYYY-MM-DD.log`

Example:
```
telemetry_2026-02-25.log
telemetry_2026-02-26.log
telemetry_2026-02-27.log
```

## Contents

Each log file contains JSON entries (one per line) with:
- **TRACE**: HTTP requests, database queries, spans
- **METRIC**: Performance metrics
- **HEADER**: Daily initialization metadata

## Usage

View today's logs:
```bash
tail -f telemetry_$(date +%Y-%m-%d).log | jq '.'
```

See [TELEMETRY.md](../../docs/TELEMETRY.md) for full documentation.

## Note

Log files are excluded from version control (`.gitignore`).
